<details><summary><박신영 문제 풀이></summary>
<p>

```python
1. CNN이란 무엇인가요?
Convolutional neural network(CNN 또는 ConvNet)는 데이터에서 직접 학습하는 디럽닝의 신경망 아키텍쳐로,
영상에서 객체, 클래스, 범주 인식을 위한 패턴을 찾을 때 특히 유용하다.
오디오, 시계열 및 신호 데이터를 분류에도 효과적이다.

2. NLP란 무엇인가요?
Natural Language Processing(NLP)(자연어 처리)은 컴퓨터가 인간의 언어를 이해, 생성,
조작할 수 있도록 해주는 인공 지능(AI)의 한 분야이다.
Siri, Cortana, Alexa와 같은 가상 도우미의 핵심 기술이다.
반복 업무 자동화, 검색 효율 향상, 검색 엔진 최적화, 대규모 문서 컬렉션 분석 및 정리, 소셜 미디어 분석 등에 활용된다.

3. RNN이란 무엇인가요?
순환 신경망(Recurrent Neural Networks, RNN)은 순차 데이터나 시계열 데이터를 이용하는 인공 신경망 유형이다.
언어 변환, 자연어 처리(nlp), 음성 인식, 이미지 캡션과 같은 순서 문제나 시간 문제에 흔히 사용한다.
순환 신경망은 학습하는 데 훈련 데이터를 활용한다.

4. GAN이란 무엇인가요?
GAN은 Generative Adversarial Networks 의 줄임말로, Generative Model의 한 종류이다.
GAN은 실제에 가까운 이미지나 사람이 쓴 것과 같은 글 등 여러 가짜 데이터들을 생성하는 모델이다.
로 다른 두 개의 네트워크를 적대적으로(adversarial) 학습시키며
실제 데이터와 비슷한 데이터를 생성(generative)해내는 모델이다.
```
