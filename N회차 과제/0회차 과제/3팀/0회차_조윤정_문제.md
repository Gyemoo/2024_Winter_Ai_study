0주차 3팀 폴더에 첨부되어 있는 'income.csv' 파일을 로드한 뒤 두 가지 문제를 해결하시오.   
아래의 코드를 그대로 붙이고 시작한다. (학습률 : 0.02, epochs=4001로 설정한다)
