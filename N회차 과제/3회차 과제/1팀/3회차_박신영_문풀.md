## 문제 1
교재에서 제시했던 XOR 함수
```python
# NAND ^ OR -> XOR
#  1 0   1 1   1 0
#  1 1   0 1   0 1

def XOR(x1,x2):
  return AND(NAND(x1,x2),OR(x1,x2))
```
