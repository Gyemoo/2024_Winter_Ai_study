### 1. 최소제곱법을 이용한 선형회귀 -> 공부시간과 성적 상관관계
```python
### SETTING

import numpy as np
import pandas as pd
import matplotlib
import matplotlib.pyplot as plt

x = [1, 2, 5, 8, 10, 12]
y = [31, 62, 71, 76, 81, 96]

xm = np.array(x).mean()
ym = np.array(y).mean()

d = 0
k = 0
for i in range(len(x)):
  d+= (xm - x[i]) * (ym - y[i])
  k+= (xm - x[i]) ** 2

a = d / k
b = ym - a * xm

n = float(input())
y_pre = a * n + b

print("%.04f시간 공부 시 성적: %.04f"%(n,y_pre))
```
